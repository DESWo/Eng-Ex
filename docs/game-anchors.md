# Familiar-game anchors

Every game is based on a popular game archetype students already know, so the
first ten seconds feel familiar and the engineering sneaks in behind it.
Trademarked names stay out of the app copy; the in-game hook uses the
archetype ("a certain bird game", "the arcade claw machine").

Three depths of anchoring:

- **deep**: the interaction itself is the familiar game (drag mechanics, art,
  win feel rebuilt around it).
- **native**: the game already IS the archetype; the level 1 copy names it.
- **hook**: the mechanic matches; the level 1 copy frames it in the familiar
  game's terms.

## Upgrade candidates (hook -> deep)

1. **Quality Gate**: an actual stamp interaction (APPROVE / REJECT) at the
   inspection desk, Papers-Please style.
2. **Don't Trip**: appliances as draggable blocks that visually fill each
   breaker column, Tetris-style.
3. **Shake Proof / Sky High**: exaggerated wobble animation and a
   block-tower art style.
4. **Assembly Line**: stations drawn as kitchen counters with order tickets.
